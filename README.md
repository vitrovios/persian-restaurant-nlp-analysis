# Market Research, Consumer Insights & Sentiment Analysis

This repository contains a public and anonymized version of a market research project conducted for the restaurant and catering industry.

The project combines customer research, consumer behavior analysis, competitor intelligence, and sentiment analysis techniques to transform customer feedback into actionable business insights.

All confidential client information, brand names, raw customer data, strategic recommendations, and final project deliverables have been removed or anonymized.

---

## Project Overview

The objective of this project was to understand customer behavior, identify market opportunities, analyze competitors, and uncover the key drivers of customer satisfaction and dissatisfaction within the restaurant and catering market.

To achieve this goal, customer reviews were analyzed using Natural Language Processing (NLP), text mining, and sentiment analysis techniques alongside broader market research methodologies.

The project demonstrates how qualitative and quantitative data can be combined to support business decision-making.
